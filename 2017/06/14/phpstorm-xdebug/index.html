xdebug+phpstorm+chrom配置记录

发表于 2017-06-14 | 分类于 编程 | 阅读次数:

PHP安装xdebug扩展配置

先安装php-xdebug（简单略过）.后修改xdebug配置如下：

PHPSTORM 配置

  1. phpstorm设置–>languages and frameworks –> php 找到php的设置 选择好php level 和 interpreter （环境的php版本和 php cli路径）
  2. php配置下面的servers： 新增项目的域名配置debugger选择xdebug
  3. php配置下面的debug： 勾选xdebug下面所有选项，xdebug端口修改为9001 和安装xdebug的配置xdebug.remote_port端口保持一致
  4. php配置下的debug下面的DBGp Proxy配置项： IDEKEY和安装xdebug的配置xdebug.idekey保持一致 ，host项目的域名，port项目的端口
  5. 保存配置后，进入工具栏的RUN ，选择 Edit configurations, 新增config 选择 php web application, server选择刚php配置的server，browser选择chrome

chrom 安装配置

  1. 安装Xdebug helper 扩展
  2. IDE key 选择 phpstorm，填写PHPSTORM，和xdebug.idekey保持一致

所有配置完成，IDE中打断点，运行debug即可
